# nodejs-complete-api
REST API node js

## Installation
- Run MongoDB -> ./Users/username/mongo/bin/mongod --dbpath=/Users/username/mongodb-data/
- npm install
- npm run dev/npm run start

## Feature:
- CRUD (user and task)
- Login -> provide jwtoken
- Logout per session & logout all session
- Middleware
- MongoDB
- Upload avatar
- Timestamps
- Filter & pagination
- SortBy (asc/desc1)
- Send email on register
